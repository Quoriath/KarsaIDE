# 🔥 Remove All Placeholders - Implementation Prompt

Hapus semua console.log dan kode simulasi di Context Menu dan Terminal dengan mengganti ke Tauri commands asli: Untuk Context Menu, ganti console.log dengan invoke('create_file', { path, content: '' }) untuk New File, invoke('create_directory', { path }) untuk New Folder, invoke('rename_path', { oldPath, newPath }) untuk Rename, invoke('delete_path', { path }) untuk Delete, dan invoke('copy_path', { source, destination }) untuk Copy/Paste. Untuk Terminal, hapus simulasi command execution dan ganti dengan real PTY: gunakan invoke('spawn_terminal', { id, shell: null }) di onMount, listen event 'terminal-output' untuk menerima output (payload adalah number[] yang harus di-decode dengan TextDecoder), dan invoke('write_to_terminal', { id, data: Array.from(encoder.encode(command + '\n')) }) untuk mengirim command. Semua commands return Promise dan throw string error jika gagal, jadi wrap dengan try-catch dan tampilkan error notification. Lihat TAURI_API_REFERENCE.md untuk signature lengkap dan contoh usage semua commands. Pastikan cleanup event listeners di onDestroy dengan memanggil unlisten() function yang dikembalikan dari listen(). Untuk file operations, selalu refresh file tree setelah operasi berhasil dengan memanggil invoke('list_directory', { path }) ulang. Untuk terminal, maintain state per terminal ID karena support multi-terminal, dan handle edge cases seperti terminal not found error.
